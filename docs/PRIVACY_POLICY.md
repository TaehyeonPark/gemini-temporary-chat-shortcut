# Privacy Policy

**Gemini Temporary Chat Shortcut** is a Chrome extension that adds a keyboard shortcut to open Gemini's temporary chat feature.

## Data Collection

This extension does **not** collect, store, or transmit any personal data or browsing information.

## Permissions

This extension requires access to `gemini.google.com` pages only, in order to:

- Listen for keyboard shortcut events
- Click the temporary chat button in the Gemini UI

No data leaves your browser. The extension has no background processes, no analytics, no tracking, and no external network requests.

## Third-Party Services

This extension does not use any third-party services or APIs.

## Changes

If this policy changes, updates will be posted in this repository.

## Contact

For questions, please open an issue on [GitHub](https://github.com/TaehyeonPark/gemini-temporary-chat-shortcut/issues).

# 개인정보 처리방침

**Gemini Temporary Chat Shortcut**은 Gemini의 임시 채팅 기능을 키보드 단축키로 여는 Chrome 확장 프로그램입니다.

## 데이터 수집

이 확장 프로그램은 개인 데이터나 브라우징 정보를 수집, 저장, 전송하지 **않습니다**.

## 권한

이 확장 프로그램은 `gemini.google.com` 페이지에만 접근하며, 다음 용도로만 사용합니다:

- 키보드 단축키 이벤트 감지
- Gemini UI의 임시 채팅 버튼 클릭

어떤 데이터도 브라우저 밖으로 전송되지 않습니다. 백그라운드 프로세스, 분석, 추적, 외부 네트워크 요청이 없습니다.

## 제3자 서비스

이 확장 프로그램은 제3자 서비스나 API를 사용하지 않습니다.

## 변경 사항

이 방침이 변경될 경우, 이 저장소에 업데이트가 게시됩니다.

## 문의

질문이 있으시면 [GitHub](https://github.com/TaehyeonPark/gemini-temporary-chat-shortcut/issues)에 이슈를 남겨주세요.
